The nano editor is an easy-to-use Unix editor.

![image](https://t1.daumcdn.net/cfile/tistory/2330CA4E528B679022)

Save: Ctrl + O

Exit : Ctrl + X

Find: Ctrl + W

Find and Replace: Ctrl + \

Specify Block: Ctrl + ^

Cut: Ctrl + K

Paste: Ctrl + U
