Simplicity of code

Typically, the following is how to treat it as an if statement:

```js
if( aaa ) {
return bbb;
} else {
return ccc;
}
```

![image](https://t1.daumcdn.net/cfile/tistory/26325E4057EC935C28)

The following is a pattern with a trinomial operator:

```js
return ( aaa ) ? bbb : ccc;
```

Next.

```js
return aaa
```
